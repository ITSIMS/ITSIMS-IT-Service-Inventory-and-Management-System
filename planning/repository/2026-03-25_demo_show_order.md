## 1. Общая структура демонстрации

**Общее время:** ~25–30 минут
**Участники:** докладчики по кандидатам, ответственный за фиксацию замечаний

| Блок | Содержание | Время |
| ---- | ---------- | ----- |
| Вводная часть | Цель демонстрации, структура, кандидаты | 2–3 мин |
| Кандидат 1: dependency_graph | Позитивные и негативный сценарии | 10 мин |
| Переход | Краткое резюме кандидата 1, переход к кандидату 2 | 1 мин |
| Кандидат 2: rbac | Позитивные и негативные сценарии | 10 мин |
| Итоговое резюме | Выводы по обоим кандидатам, вопросы | 3–5 мин |
