## Описание

Необходимо грамотно заполнить документ трассировки между требованиями к программному обеспечению (SWR) и требованиями к системе (SR).

**Файл для создания/редактирования:** `artifacts/4_SR_to_SWR.md`

**Входные данные:**
- [Требования к системе (SR)](../../artifacts/3_system_requirements.md)
- [Требования к ПО (SWR)](../../artifacts/5_software_requirements.md) — созданный Вальковцом
- [Пример трассировки TS-SR](../../artifacts/2_TS_to_SR.md) — для референса

**Что нужно сделать:**
- Создать/заполнить матрицу трассировки SR → SWR
- Убедиться, что каждое SR-требование покрыто минимум одним SWR-требованием
- Обеспечить полноту покрытия (все SR должны быть покрыты)
- Проверить корректность связей между требованиями

**Зависимости:**
- Задача зависит от завершения задачи Вальковца по созданию первой версии SWR
