# Зафиксировать принцип привязки тестов к требованиям

**Создана:** 2026-02-22
**Ответственный:** Зацепилин
**Нужно сделать до:** 2026-02-28

## Описание

Добавить в регламент верификации правило: тест-кейсы привязаны к идентификаторам требований; при изменении требований пересматриваются тесты; при изменении только кода тесты не переписываются.
