## Назначение директории

Директория `tools/` содержит **вспомогательные инструменты и скрипты** для работы с проектом.

**Что здесь хранится:**
- Скрипты для конвертации документов
- Инструменты автоматизации
- Инструкции по использованию инструментов
- Вспомогательные ресурсы и конфигурационные файлы

**Что здесь НЕ хранится:**
- Исходный код самой системы ITSIMS
- Временные файлы
- Рабочие документы проекта



## Структура директории

```
tools/
├── README.md                   # Данный документ
└── printing/                   # Инструменты для печати документов
    ├── prepare_for_print.sh     # Подготовка одного документа к печати
    ├── prepare_all_for_print.sh # Подготовка всех документов к печати
    ├── instructions/            # Инструкции по работе с инструментами
    │   ├── markdown_to_pdf_guide.md
    │   └── printing_instructions.md
    └── resources/               # Ресурсы для конвертации
        ├── default_table_width.lua
        └── listings-setup.tex
```



## Краткое описание инструментов

**`prepare_for_print.sh`** — подготовка документа к печати

**`prepare_all_for_print.sh`** — подготовка всех документов

**Назначение:** Конвертирует один (соответственно все) Markdown документ(-ы) в PDF с правильным форматированием.

Подробные инструкции по работе с инструментами печати:

- [Руководство по конвертации Markdown в PDF](./printing/instructions/markdown_to_pdf_guide.md)
- [Инструкции по печати документов](./printing/instructions/printing_instructions.md)



## Правила работы с инструментами

### Исполняемые права

Все скрипты должны быть исполняемыми:

```bash
chmod +x tools/printing/prepare_for_print.sh
chmod +x tools/printing/prepare_all_for_print.sh
```

### Документирование инструментов

К каждому новому инструменту должна быть создана инструкция:

```
tools/
├── new_tool.sh                      # Новый скрипт
└── instructions/                    
    └── new_tool_guide.md            # Инструкция к скрипту
```

### Размещение ресурсов

Конфигурационные файлы и ресурсы размещаются в соответствующих подпапках:

```
tools/
└── printing/
    └── resources/                   # Ресурсы для печати
        ├── default_table_width.lua  # Pandoc фильтры
        └── listings-setup.tex       # LaTeX настройки
```
